Quick start

Up and running in three commands.

~ fgm

$ fgm install latest

Resolving go 1.26.1...

Downloading go1.26.1.linux-amd64.tar.gz...

Verifying checksum...

Extracting...

✓ Installed go 1.26.1

$ fgm use latest

✓ Using go 1.26.1

$ go version

$ fgm list

Commands

A small, focused command set.

fgm install <version>
or fgm in <version>

Download a Go version without activating it.

fgm use <version>

Install if needed, then activate. Updates shims instantly.

fgm uninstall <version>
or fgm un <version>

Remove a previously installed version.

fgm list
or fgm ls

List installed versions, marking the current one.

fgm current

Print the currently active Go version.

fgm prune

Remove cached downloads and the manifest cache.

fgm completion <shell>

Generate completion script (bash, zsh, fish, powershell).

--verbose, -v

Verbose diagnostics — manifest cache state, archive details, platform info.

Version formats

latest Latest stable release.
1.25 Latest patch of Go 1.25 (e.g. 1.25.8).
1.25.5 Pinned exact release.

How it works

No magic — just three things.

Download & verify. fgm fetches the official Go release manifest from go.dev/dl (cached locally for 15 minutes), pulls the right archive for your platform, and verifies its SHA-256 before extraction.

Extract & track. Each toolchain lives in ~/.fgm/versions/<version>/. The active version is stored as plain text in ~/.fgm/current-version. Multiple versions coexist without conflict.

Resolve at runtime. go and gofmt shims in ~/.fgm/bin/ read current-version on every invocation and exec the right binary. No eval, no source, no shell hooks.
